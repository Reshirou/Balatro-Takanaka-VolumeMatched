# Balatro OST replacement pack in the style of Masayoshi Takanaka!

**How to use:**

1: Prepare Balatro for modding, if you have never done this before I would suggest looking for an up to date guide on the process!

2: Download [Penumbra](https://github.com/lord-ruby/Penumbra) and paste Penumbra-main in your mods folder:

  **%appdata%/Balatro/Mods**

3: If you aren't sure how to add music to Penumbra, open the Penumbra-main folder, open the assets folder, and place the "sounds" folder from within either Takanaka folder DIRECTLY into that assets folder. If there is already a "sounds" folder, you can safely merge them.


4: This should already work, but if not you can check the Penumbra mod config within Balatro and check all the music tracks with Takanaka in their name!


**If the game crashes on launch and you've followed every prior step, this is likely due to incompatibility with steam modded. Trying an older (or newer) version will likely fix this issue.**
